Much like last week, you have a number of CSS challenges before you. Follow along and by the time you're done, you'll have completed your first all-css page layout. (It's going to be a single-column layout, with a horizontal nav.)

Tasks

  1. Choose a font, font-size, and color. Consider if headers should be distinct from lists or body text. Set the body background color to be different than the #wrapper div's background color.
  2. Choose an overall width (hint: set this on the wrapper div), and make the page auto-center even as the browser size changes. (hint: use margin: 20px auto or something similar)
  3. Adjust padding and line-height until you're happy with the general typography of your document.
  4. Set list-style-type: none and padding: 0 on the ul element inside the nav element.
  5. Set float: left on the li elements inside the nav element. Set a width on each of these as well. Note that you can't just write a rule for all li's, or the numbered list of tasks will get messed up.
  6. You may notice your layout has gotten a little weird. That's float for you! Fix it up by setting clear: left on the "content" div. You might also want to try setting a height on the nav.
  7. Set display: block on the a elements inside the nav. Give them a different background color. Not happy with the text alignment? Set text-align: center on them, and give them a little side margin to seperate them from each other.
  8. Use the hover pseudo-class to make the nav elements react to the mouse cursor passing over them. Your choice as to how they react. A couple of ideas: change color, change border, change background color, etc.
  9. Space out the numbered task li elements from each other a bit, using top and/or bottom margins.
  10. Make the <code> elements look better!
